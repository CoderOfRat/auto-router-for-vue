# auto-router-vue

### 项目介绍

该项目是基于vue-cli4的 vue router 配置对象自动生成的小模板，仅供学习参考

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
